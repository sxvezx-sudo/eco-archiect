## Image Assets

`https://www.figma.com/api/mcp/asset/...` URLs **expire after 7 days**. When images stop loading:

1. Use the Figma MCP tool: call `get_design_context` with the relevant node ID and file key above
2. Copy the fresh asset URLs from the response
3. Update the `src` attributes in `components/navbar.html`, `components/footer.html`, or the relevant HTML file
